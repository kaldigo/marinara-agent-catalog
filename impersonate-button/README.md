# Impersonate Button

Package-era rewrite of the legacy Impersonate Button extension.

This package adds four bridge-managed composer quick actions for dry-run persona
generation helpers. Composer mounting and generation lifecycle behavior are
routed through `_mari-bridge`.

## Features

- Dry-run impersonate generation into the current draft.
- Continue draft and Inner State generation modes.
- Restore the last saved guidance text per chat.
